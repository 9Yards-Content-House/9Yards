# 9Yards Multi-Site Ecosystem: Parent Hub

Welcome to the central repository for the **9Yards Parent Conglomerate Site**. This project serves as the digital "Airport Terminal" for Uganda's premier creative ecosystem, routing traffic to our specialized subsidiaries in content, food, film, and gear.

## Project Vision

To build a world-class corporate hub that establishes 9Yards as a forward-thinking creative leader in East Africa. This site focuses on **credibility, ecosystem synergy, and high-conversion navigation.**

---

## Information Architecture

The parent site is designed for maximum clarity and speed:

* **Home:** The "Wow" factor. Cinematic hero and subsidiary power cards.
* **About:** Our story, philosophy, and the "Building Uganda" mission.
* **Companies:** Deep-dive into our 4 business units with direct links.
* **Team & Careers:** Humanizing the brand and attracting the best talent.
* **Contact:** Centralized inquiry hub with Netlify-powered routing.

---

## Tech Stack (Phase 1)

* **Frontend:** Semantic HTML5, CSS3 (Tailwind CSS for utility), Vanilla JavaScript.
* **Animations:** AOS (Animate on Scroll) for a premium feel.
* **Hosting:** Netlify (Global CDN).
* **Forms:** Netlify Forms (Zero-backend contact management).
* **Performance:** WebP Image Optimization & Lazy Loading.


## Brand Standards

All development must adhere to the **9Yards Brand Manual**:

* **Primary Blue:** `#212282` (Trust & Authority)
* **Primary Orange:** `#E6411C` (Energy & Action)
* **Typography:** Open Sans (Bold for headings, Regular for body)

---

## Getting Started

To contribute to the build or run locally:

1. **Clone the repo:**
```bash
git clone https://github.com/9Yards-Content-House/9Yards.git

```


2. **Open in your preferred environment:**
No build step required for Phase 1. Simply open `index.html` via Live Server.
3. **Deployment:**
Push to `main` to trigger an automatic deploy to **Netlify**.
